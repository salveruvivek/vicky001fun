# Git002 — Git Practice Project

A small collection of files for practicing Git commands.

## Files

| File             | Description                          |
|------------------|--------------------------------------|
| `login_page.py`  | Simple console login program         |
| `register.py`    | Register a new user                  |
| `calculator.py`  | Basic add/subtract/multiply/divide   |
| `index.html`     | Simple HTML page                     |
| `.gitignore`     | Files Git should ignore              |

## Common Git commands to practice

```bash
git status                  # what changed
git add <file>              # stage a file
git add .                   # stage everything
git commit -m "message"     # save a snapshot
git log --oneline           # view history
git diff                    # see unstaged changes
git branch new-feature      # create a branch
git checkout new-feature    # switch to it
git merge new-feature       # merge it back
```
