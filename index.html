🌿 Git Commands Reference

A quick, searchable cheat sheet of everyday Git commands

1. Setup & Configuration

git config --global user.name "Name"Set your name
git config --global user.email "you@mail.com"Set your email
git config --listView all settings
git --versionCheck Git version

2. Starting a Repository

git initCreate a new repo here
git clone <url>Copy a remote repo locally
git clone <url> <folder>Clone into a specific folder

3. Basic Snapshotting

git statusShow changed/staged files
git add <file>Stage one file
git add .Stage everything
git reset <file>Unstage a file (keep changes)
git commit -m "message"Commit staged changes
git commit -am "message"Stage tracked files + commit
git commit --amendEdit the last commit

4. Viewing Changes & History

git diffUnstaged changes
git diff --stagedStaged changes
git logFull commit history
git log --onelineCompact history
git log --oneline --graph --allVisual branch graph
git show <commit>Details of a commit
git blame <file>Who changed each line

5. Branching

git branchList branches
git branch <name>Create a branch
git checkout <name>Switch to a branch
git checkout -b <name>Create + switch
git switch -c <name>Create + switch (newer)
git merge <name>Merge branch into current
git branch -d <name>Delete a branch
git branch -m <old> <new>Rename a branch

6. Remote Repositories

git remote -vList remotes
git remote add origin <url>Link a remote
git pushPush commits
git push -u origin mainPush + set upstream (first time)
git pullFetch + merge from remote
git fetchDownload changes (no merge)

7. Undoing Things

git restore <file>Discard unstaged changes
git restore --staged <file>Unstage a file
git reset --soft HEAD~1Undo last commit, keep staged
git reset --hard HEAD~1Undo last commit, DISCARD changes
git revert <commit>New commit that undoes one
git clean -fdDelete untracked files & folders

8. Stashing

git stashSave uncommitted changes
git stash listList stashes
git stash popRestore + remove newest stash
git stash applyRestore but keep the stash
git stash clearDelete all stashes

9. Tags

git tagList tags
git tag -a v1.0 -m "Release"Create an annotated tag
git push origin v1.0Push a tag
git push origin --tagsPush all tags

10. Inspecting & Recovering

git diff branchA..branchBCompare two branches
git shortlog -snCommit count per author
git reflogHistory of HEAD (recover commits)
